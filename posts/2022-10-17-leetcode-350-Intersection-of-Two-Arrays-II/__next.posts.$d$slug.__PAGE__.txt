<h3>문제</h3>
<blockquote>
<p><a href="https://leetcode.com/problems/intersection-of-two-arrays-ii/description/">350. Intersection of Two Arrays II 문제 링크</a></p>
</blockquote>
<p>배열 <code>nums1</code>, <code>nums2</code>의 교집합을 구하시오.</p>
<pre><code class="hljs language-text"># Example 1:
Input: nums1 = [1,2,2,1], nums2 = [2,2]
Output: [2,2]
</code></pre>
<hr>
<h3>처음 작성한 풀이</h3>
<pre><code class="hljs language-python"><span class="hljs-keyword">def</span> <span class="hljs-title function_">intersect</span>(<span class="hljs-params">self, nums1: <span class="hljs-type">List</span>[<span class="hljs-built_in">int</span>], nums2: <span class="hljs-type">List</span>[<span class="hljs-built_in">int</span>]</span>) -> <span class="hljs-type">List</span>[<span class="hljs-built_in">int</span>]:
  result = []
  <span class="hljs-keyword">for</span> n <span class="hljs-keyword">in</span> nums2:
    <span class="hljs-keyword">if</span> n <span class="hljs-keyword">in</span> nums1:
      nums1.remove(n)
      result.append(n)
  <span class="hljs-keyword">return</span> result
</code></pre>
<p>하고 보니 <code>remove</code> 함수의 시간복잡도가 O(n)이기 때문에 결과적으로 O(n^2)가 되어 좋지 않은 풀이었습니다. 😓</p>
<h3>최종 Solution</h3>
<pre><code class="hljs language-python"><span class="hljs-keyword">def</span> <span class="hljs-title function_">intersect</span>(<span class="hljs-params">self, nums1: <span class="hljs-type">List</span>[<span class="hljs-built_in">int</span>], nums2: <span class="hljs-type">List</span>[<span class="hljs-built_in">int</span>]</span>) -> <span class="hljs-type">List</span>[<span class="hljs-built_in">int</span>]:
  nums1.sort()
  nums2.sort()
  i = <span class="hljs-number">0</span>
  j = <span class="hljs-number">0</span>
  result = []
  <span class="hljs-keyword">while</span> i &#x3C; <span class="hljs-built_in">len</span>(nums1) <span class="hljs-keyword">and</span> j &#x3C; <span class="hljs-built_in">len</span>(nums2):
    <span class="hljs-keyword">if</span> nums1[i] &#x3C; nums2[j]:

</code></pre>
