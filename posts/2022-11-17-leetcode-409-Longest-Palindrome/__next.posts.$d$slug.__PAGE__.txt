<h3>문제</h3>
<blockquote>
<p><a href="https://leetcode.com/problems/longest-palindrome/?envType=study-plan&#x26;id=level-1">409. Longest Palindrome 문제 링크</a></p>
</blockquote>
<p>가장 긴 팰린드롬을 구하시오!</p>
<blockquote>
<p>팰린드롬(palindrome)?</p>
</blockquote>
<p>팰린드롬은 역순으로 읽어도 같은 것을 말합니다.
<br>
예를 들어 <code>bob</code> 과 같이 거꾸로 해도 <code>bob</code>인 단어 입니다.
<br>
(한국 예능에서는 이효리, 이혜리 가 있었죠..)</p>
<pre><code class="hljs language-text"># Example 1:
Input: s = "abccccdd"
Output: 7
Explanation: One longest palindrome that can be built is "dccaccd", whose length is 7.
</code></pre>
<p>위의 예제에서 만들 수 있는 가장 긴 팰린드롬 단어는 dccaccd 입니다.
<br>
(a 를 b 로 대체하여 dccbccd 로 할 수도 있죠.)</p>
<h3>최종 Solution</h3>
<ol>
<li>글자 수가 짝수인 경우 그 단어는 무조건 사용할 수 있기에 포함!
  <br>
</li>
</ol>
<p>(위의 예제에서는 c와 d가 있습니다.)
<br>
딱 떨어지는 개수가 아닌 aaa 라는 단어가 있어도! aa 만 추출하면 되니 이도 포함되어야 합니다. (여기서 한번 틀림... 🫠 )
2. 그리고 한 글자 이더라도 가장 중앙에 위치한다면 아무 문제가 없습니다.
<br>
(위의 예제의 중앙의 a와 같습니다.)</p>
<p>고로 정답 == 짝수개의 단어 + 한 글자(IF 한 글자가 있다면)</p>
<pre><code class="hljs language-python"><span class="hljs-keyword">def</span> <span class="hljs-title function_">longestPalindrome</span>(<span class="hljs-params">self, s: <span class="hljs-built_in">str</span></span>) -> <span class="hljs-built_in">int</span>:
  dic = collections.Counter(s)
  result = <span class="hljs-number">0</span>
  odd = <span class="hljs-literal">False</span>
  <span class="hljs-keyword">for</span> letter <span class="hljs-keyword">in</span> dic:

    <span class="hljs-keyword">if</span> dic[letter] % <span class="hljs-number">2</span> == <span class="hljs-number">1</span>:
      odd = <span class="hljs-literal">True</span>
  <span class="hljs-keyword">return</span> result + odd
