<h3>문제</h3>
<blockquote>
<p><a href="https://leetcode.com/problems/binary-tree-level-order-traversal/?envType=study-plan&#x26;id=level-1">102. Binary Tree Level Order Traversal 문제 링크</a></p>
</blockquote>
<p>Binary Tree → Level 순서로 탐색하기!</p>
<blockquote>
<p>🤔 Binary Tree(이진 트리)?
<br>
자식 노드가 최대 두 개인 노드들로 구성된 트리 입니다.</p>
</blockquote>
<pre><code class="hljs language-text"># Example 1:
Input: root = [3,9,20,null,null,15,7]
Output: [[3],[9,20],[15,7]]
</code></pre>
<p><img src="https://user-images.githubusercontent.com/93169519/230064070-fcb82fc7-00b3-4f10-b362-3692eba2fc01.png" alt="image"></p>
<p>문제의 핵심은 <code>Level</code> 순서로 진행 된다는 점!
<br>
🤔 만약에 위의 예제에서 9의 자식노드 [10, 4]가 추가로 존재한다면?
<br>
→ Output: [[3],[9,20],[10,4,15,7]] 으로 된다는 것!</p>
<h3>최종 Solution</h3>
<p>각 노드의 왼쪽 오른쪽에 있는 것을 <code>Level</code> 별로 하나로 합쳐야 됩니다.
<br>
그를 위해 노드를 하나의 배열로 모아놓은 <code>level</code> 변수를 이용해서 반복하여 풀어냈습니다.</p>
<pre><code class="hljs language-python"><span class="hljs-keyword">def</span> <span class="hljs-title function_">levelOrder</span>(<span class="hljs-params">self, root: <span class="hljs-type">Optional</span>[TreeNode]</span>) -> <span class="hljs-type">List</span>[<span class="hljs-type">List</span>[<span class="hljs-built_in">int</span>]]:
  ans, level = [], [root]
  <span class="hljs-keyword">while</span> root <span class="hljs-keyword">and</span> level:
    ans.append([node.val <span class="hljs-keyword">for</span> node <span class="hljs-keyword">in</span> level])
    LRs = [(node.left, node.right) <span class="hljs-keyword">for</span> node <span class="hljs-keyword">in</span> level]
    level = [leaf <span class="hljs-keyword">for</span> LR <span class="hljs-keyword">in</span> LRs <span class="hljs-keyword">for</span> leaf <span class="hljs-keyword">in</span> LR <span class="hljs-keyword">if</span> leaf]
  <span class="hljs-keyword">return</span> ans
</code></pre>
<p>while 문 부분의 <code>level</code> 부분만 보면 아래와 같습니다.</p>

LRs = []
<span class="hljs-keyword">for</span> node <span class="hljs-keyword">in</span> level:
  LRs.append((node.left, node.right));

<span class="hljs-comment"># 2. (left, right)로 묶인 것을 배열 형태로 풀어 등록하기.</span>
level = []
<span class="hljs-keyword">for</span> LR <span class="hljs-keyword">in</span> LRs:
