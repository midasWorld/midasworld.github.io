<h3>문제</h3>
<blockquote>
<p><a href="https://leetcode.com/problems/binary-tree-level-order-traversal/?envType=study-plan&#x26;id=level-1">104. Maximum Depth of Binary Tree 풀기 완료 문제 링크</a></p>
</blockquote>
<p>Binary Tree → 최대 깊이 구하기!</p>
<blockquote>
<p>🤔 Binary Tree(이진 트리)?
<br>
자식 노드가 최대 두 개인 노드들로 구성된 트리 입니다.</p>
</blockquote>
<p><img src="https://user-images.githubusercontent.com/93169519/230863009-94cf7a63-7198-4a16-b890-40a5710705d2.png" alt="image"></p>
<pre><code class="hljs language-text"># Example 1:
Input: root = [3,9,20,null,null,15,7]
Output: 3
</code></pre>
<p>사진을 보면 알 수 있듯이 가장 깊은건  15, 7(3 level)이므로 정답은 3입니다.</p>
<h3>최종 Solution</h3>
<p>결과적으로 가장 깊은 노드까지 반복적으로 내려가며 반복하는 방법 밖에 없는 것 같습니다.</p>
<pre><code class="hljs language-python"><span class="hljs-keyword">def</span> <span class="hljs-title function_">maxDepth</span>(<span class="hljs-params">self, root: <span class="hljs-type">Optional</span>[TreeNode]</span>) -> <span class="hljs-built_in">int</span>:
  <span class="hljs-keyword">if</span> <span class="hljs-keyword">not</span> root: <span class="hljs-keyword">return</span> <span class="hljs-number">0</span>

  ans, level = <span class="hljs-number">0</span>, [root]
  <span class="hljs-keyword">while</span> level:
    ans += <span class="hljs-number">1</span>
    LRs = []
    <span class="hljs-keyword">for</span> node <span class="hljs-keyword">in</span> level:
      <span class="hljs-keyword">if</span> node.left:
        LRs.append(node.left)
      <span class="hljs-keyword">if</span> node.right:
        LRs.append(node.right)
    level = LRs
  <span class="hljs-keyword">return</span> ans
</code></pre>
<p>일단 빈 배열로 들어오면 0을 반환 합니다.
<br>
그리고 나서는 한 단계씩 '비어 있지 않은 하위 노드(left, right)'를 변수 <code>level</code>로 전달해서 계속 반복하는 것 뿐입니다.</p>
<p>아래의 <code>dfs</code>로 풀어낸 것과 거의 똑같은 속도가 측정되었습니다. 🥸</p>
<h2>Best Solution</h2>
<p>이걸 <code>dfs</code>로 풀어내니 코드가 이렇게 짧아지는 구나... 싶었습니다. 🤔</p>

  <span class="hljs-keyword">def</span> <span class="hljs-title function_">dfs</span>(<span class="hljs-params">root, depth</span>):
    <span class="hljs-keyword">if</span> <span class="hljs-keyword">not</span> root: <span class="hljs-keyword">return</span> depth
