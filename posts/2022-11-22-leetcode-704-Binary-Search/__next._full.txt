<h3>문제</h3>
<blockquote>
<p><a href="https://leetcode.com/problems/binary-search/">704. Binary Search 풀기 완료 문제 링크</a></p>
</blockquote>
<pre><code class="hljs language-text"># Example 1:
Input: nums = [-1,0,3,5,9,12], target = 9
Output: 4
Explanation: 9 exists in nums and its index is 4
</code></pre>
<p>nums 배열에서 타겟의 숫자는 9의 인덱스는 <code>4</code>이므로 <code>4</code>가 반환 됩니다.</p>
<h3>✅해결 방법 &#x26; 최종 Solution</h3>
<p>문제 자체가 Binary Search 이므로 이 방법으로 풀었습니다.</p>
<pre><code class="hljs language-python"><span class="hljs-keyword">class</span> <span class="hljs-title class_">Solution</span>:
    <span class="hljs-keyword">def</span> <span class="hljs-title function_">search</span>(<span class="hljs-params">self, nums: <span class="hljs-type">List</span>[<span class="hljs-built_in">int</span>], target: <span class="hljs-built_in">int</span></span>) -> <span class="hljs-built_in">int</span>:
        start, end = <span class="hljs-number">0</span>, <span class="hljs-built_in">len</span>(nums) - <span class="hljs-number">1</span>
        <span class="hljs-keyword">while</span> start &#x3C;= end:
            mid = (end - start) // <span class="hljs-number">2</span> + start
            <span class="hljs-keyword">if</span> nums[mid] == target:
                <span class="hljs-keyword">return</span> mid
            <span class="hljs-keyword">elif</span> nums[mid] &#x3C; target:
                start = mid + <span class="hljs-number">1</span>
            <span class="hljs-keyword">else</span>:
                end = mid - <span class="hljs-number">1</span>
        <span class="hljs-keyword">return</span> -<span class="hljs-number">1</span>
</code></pre>
<p>위의 예제로 순서대로 구해보면?</p>
<pre><code class="hljs language-shell"><span class="hljs-meta prompt_"># </span><span class="bash">Example</span>

1. ❎ start = 0, end = 5, mid = 2, nums[mid] = 3
2. ✅ start = 3, end = 5, mid = 4, nums[mid] = 9 (정답 끗!) 
</code></pre>
