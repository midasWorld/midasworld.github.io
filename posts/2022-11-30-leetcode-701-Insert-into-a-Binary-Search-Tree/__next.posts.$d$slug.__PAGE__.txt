<p><a href="https://leetcode.com/problems/insert-into-a-binary-search-tree/description/?envType=study-plan&#x26;id=data-structure-i">701. Insert into a Binary Search Tree 문제 링크</a></p>

<p>BST 에 주어진 값의 노드 추가하기!</p>
<p><img src="https://user-images.githubusercontent.com/93169519/231196434-8bfbeaa6-0397-4a63-b34f-0938388820ee.png" alt="image"></p>

Input: root = [4,2,7,1,3], val = 5

</code></pre>
<p>먼저 Binary Search Tree 특성에 대해 알아보자면...</p>
<blockquote>
<p><strong>🤔 Binary Search Tree 특성</strong></p>
<ol>
<li>모든 노드의 키는 중복되지 않는다. (중복되는 숫자가 없다!)</li>
<li>루트 노드의 왼쪽 서브 트리는 해당 노드의 키보다 작은 값을 갖는다. (root > root.left)</li>
<li>루트 노드의 오른쪽 서브 트리는 해당 노드의 키보다 큰 값을 갖는다. (root &#x3C; root.right )</li>
</ol>
</blockquote>
<p>즉슨 root.left &#x3C; root &#x3C; root.right 로 이루어져 있습니다.
<br>
고로 값의 크기에 따라 적절한 곳에 노드를 만들어 연결해주면 됩니다.</p>
<h3>✅ 최종 Solution</h3>
<p>중복되지 않고 값에 따라 들어갈 위치가 정해져 있기 때문에 걱정없이 재귀를 사용했습니다.
<br>
빈 상태로 받으면 노드를 생성해줘야 되기 때문에 TreeNode 를 만들어 반환되도록 하고,
<br>
비교 연산자에 의해서 각 left or right에 하위 노드에 들어갈 수 있도록 반복해주면 끝입니다.</p>
<pre><code class="hljs language-python"><span class="hljs-keyword">def</span> <span class="hljs-title function_">insertIntoBST</span>(<span class="hljs-params">self, root: <span class="hljs-type">Optional</span>[TreeNode], val: <span class="hljs-built_in">int</span></span>) -> <span class="hljs-type">Optional</span>[TreeNode]:
  <span class="hljs-keyword">if</span> <span class="hljs-keyword">not</span> root: 
    <span class="hljs-keyword">return</span> TreeNode(val)
  
  <span class="hljs-keyword">if</span> root.val > val:
    root.left = <span class="hljs-variable language_">self</span>.insertIntoBST(root.left, val)
  <span class="hljs-keyword">else</span>:
    root.right = <span class="hljs-variable language_">self</span>.insertIntoBST(root.right, val)

  <span class="hljs-keyword">return</span> root
