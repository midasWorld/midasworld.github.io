<h3>문제</h3>
<blockquote>
<ol start="98">
<li>Validate Binary Search Tree 문제 링크](<a href="https://leetcode.com/problems/validate-binary-search-tree/description/?envType=study-plan&#x26;id=data-structure-i">https://leetcode.com/problems/validate-binary-search-tree/description/?envType=study-plan&#x26;id=data-structure-i</a>)</li>
</ol>
</blockquote>
<p>올바른 BST 인지 판별하시오!</p>
<p><img src="https://user-images.githubusercontent.com/93169519/231334574-9d010410-986b-4110-88c2-f16b097bfaa1.png" alt="image"></p>
<pre><code class="hljs language-text"># Example 1:
Input: root = [2,1,3]
Output: true
</code></pre>
<p>2의 왼쪽엔 작은 1, 오른쪽은 큰 3이 있으므로 올바릅니다. 👍</p>
<hr>
<p><img src="https://user-images.githubusercontent.com/93169519/231334559-6efb3ccc-5fb3-4ab2-9f25-185499279936.png" alt="image"></p>
<pre><code class="hljs language-text"># Example 1:
Input: root = [5,1,4,null,null,3,6]
Output: false
Explanation: The root node's value is 5 but its right child's value is 4.
</code></pre>
<p>5의 오른쪽 노드인 4가 5보다 작으므로 올바르지 않습니다!</p>
<h3>✅ 해결 방법 찾아보기</h3>
<p><img src="https://user-images.githubusercontent.com/93169519/231334542-d8905517-3839-4d60-b145-c6c527bb7a80.png" alt="image"></p>
<pre><code class="hljs language-text"># Example 1:
Input: root = [5,4,6,null,null,3,7]
Output: false
</code></pre>
<p>테스트 케이스에서 위에서 실패 했습니다.
<br>
저게 왜 문제지...? 했는데 Discussion 란에 저와 같이 질문한 사람이 있었습니다.
<br></p>
<blockquote>
<p>❗️ 문제는 Root의 5의 오른쪽 노드에는 모두 5보다 큰 수만 있어야 되는 것입니다.
<br>
→ 5의 오른쪽 하위 노드들 중에 3이 있는 것이 문제인 것이죠!</p>
</blockquote>
<p>이를 위해 lower, upper 두 개의 변수를 사용했습니다.
<br></p>
<ul>
<li>오른쪽 노드를 검사할 때는 루트보다 커야 되므로 lower 를 갱신해주고,</li>
<li>왼쪽 노드를 검사할 때는 루트보다 작아야 하므로 upper 를 갱신해주는 것이죠!
<br>
</li>
<li>(lower &#x3C; node.val &#x3C; upper 로 검사 할 수 있도록 합니다.)</li>
</ul>
<p>이를 코드로 구현해 보면 아래와 같습니다.</p>
