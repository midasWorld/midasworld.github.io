<h2>사용자 생성 및 권한 부여하기</h2>
<p>아래의 명령어들을 통해 생성한 데이터베이스를 사용하실 수 있습니다.</p>
<h3>😐 사용자 생성</h3>
<pre><code class="hljs language-shell"><span class="hljs-meta prompt_">#</span><span class="bash"><span class="hljs-comment">## 생성 ###</span></span>
use mysql;

create user '사용자'@'host' identified by '비밀번호';
<span class="hljs-meta prompt_">
# </span><span class="bash">ex1) 내부 접근을 허용하는 사용자 추가</span>
create user 'test'@'localhost' identified by '0000';
<span class="hljs-meta prompt_">
# </span><span class="bash">ex2) 외부 접근을 허용하는 사용자 추가</span>
create user 'test'@'%' identified by '0000';
<span class="hljs-meta prompt_">
# </span><span class="bash">ex3) 특정 ip만 접근을 허용하는 사용자 추가</span>
create user 'test'@'123.456.789.100' iden****tified by '0000';
<span class="hljs-meta prompt_">
# </span><span class="bash">ex4) 특정 ip 대역을 허용하는 사용자 추가</span>
create user 'test'@'%' identified by '0000';
</code></pre>
<h3>🫥 사용자 제거</h3>
<pre><code class="hljs language-shell"><span class="hljs-meta prompt_">#</span><span class="bash"><span class="hljs-comment">## 제거 ###</span></span>
drop user '사용자';
<span class="hljs-meta prompt_"># </span><span class="bash">or</span>
delete from user where user='사용자';
<span class="hljs-meta prompt_">
# </span><span class="bash">example</span>
drop user 'test'@'localhost';

//== 비밀변호 변경 ==//
alter user '사용자'@'localhost' identified by '비밀번호';
</code></pre>
<h3>😎 GRANT 를 이용한 사용자 권한 부여</h3>
<pre><code class="hljs language-shell"><span class="hljs-meta prompt_"># </span><span class="bash">모든 데이터베이스의 모든 테이블에 모든 권한을 줌</span>
grant all privileges on *.* to '사용자'@'localhost';
<span class="hljs-meta prompt_">
# </span><span class="bash">특정 데이터베이스의 모든 테이블에 모든 권한을 줌</span>
grant all privileges on DB이름.* to '사용자'@'localhost';
<span class="hljs-meta prompt_">
# </span><span class="bash">특정 데이터베이스의 특정 테이블에 모든 권한을 줌</span>
grant all privileges on DB이름.테이블명 to '사용자'@'localhost';
<span class="hljs-meta prompt_">
# </span><span class="bash">특정 데이터베이스의 특정 테이블에 <span class="hljs-keyword">select</span> 권한을 줌</span>
grant select on DB이름.테이블명 to '사용자'@'localhost';
<span class="hljs-meta prompt_">
# </span><span class="bash">특정 데이터베이스의 특정 테이블에 <span class="hljs-keyword">select</span>, insert 권한을 줌</span>
grant select, insert on DB이름.테이블명 to '사용자'@'localhost';
<span class="hljs-meta prompt_">
# </span><span class="bash">특정 데이터베이스의 특정 테이블의 컬럼1과 컬럼2의 update 권한을 줌</span>
grant update(컬럼1, 컬럼2) on DB이름.테이블명 to '사용자'@'localhost';
</code></pre>
