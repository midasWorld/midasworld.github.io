<pre><code class="hljs language-javascript"><span class="hljs-keyword">const</span> <span class="hljs-title function_">plus</span> = (<span class="hljs-params">a, b</span>) => {
  <span class="hljs-keyword">return</span> a + b;

</code></pre>

<li>함수의 매개변수로 전달</li>

<li>동일 비교 대상</li>
</ul>

<h2>고차 함수 (Higher-order function)</h2>
<ul>
<li>인자를 함수를 받거나 (콜백함수),</li>
<li>함수를 반환하는 함수를 고차함수라 부릅니다.</li>
</ul>
<pre><code class="hljs language-javascript"><span class="hljs-keyword">function</span> <span class="hljs-title function_">calculate</span>(<span class="hljs-params">a, b, action</span>) {
  <span class="hljs-keyword">let</span> result = <span class="hljs-title function_">action</span>(a, b);
  <span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-string">`(<span class="hljs-subst">${a}</span>, <span class="hljs-subst">${b}</span>)'s plus result is <span class="hljs-subst">${result}</span>`</span>);
  <span class="hljs-keyword">return</span> result;
}
</code></pre>
