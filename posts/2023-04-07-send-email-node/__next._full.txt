<h3>2. <code>/service/email.ts</code> 구현</h3>
<pre><code class="hljs language-shell">import nodemailer from "nodemailer";

export type EmailData = {
  from: string;
  subject: string;
  message: string;
};

const transporter = nodemailer.createTransport({
  host: "smtp.gmail.com",
  port: 465,
  secure: true,
  auth: {
    user: process.env.EMAIL_USER,
    pass: process.env.EMAIL_PASSWORD,
  },
});

export async function sendEmail({ from, subject, message }: EmailData) {
  return transporter.sendMail({
    from,
    to: process.env.EMAIL_USER,
    subject,
    html: `
    &#x3C;div style="background-color: #f8f9f9; padding: 2.5em">
      &#x3C;h1>${subject}&#x3C;/h1>
      &#x3C;p>${message}&#x3C;/p>
      &#x3C;h3>from: &#x3C;/span>${from}&#x3C;/h3>
    &#x3C;/div>
    `,
  });
}
</code></pre>
<p><code>nodemailer</code> 관련된 코드를 모아놓고 관리하기 위해 위와 같이 만들어줬습니다.</p>

<p><code>ContactForm</code>는 클라이언트 컴포넌트이기 때문에 <code>fs</code> 사용할 수가 없는 것이었습니다. ❗️

<p>그래서 <code>fetch</code>를 사용해서 서버 API로 전송 후에 서버측에서 사용되도록 하였습니다.</p>
