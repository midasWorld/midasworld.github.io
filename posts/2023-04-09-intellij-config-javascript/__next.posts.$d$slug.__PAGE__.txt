<h2>curly braces ('{}') for authformat</h2>
<h3>관련 링크: <a href="https://stackoverflow.com/questions/41903215/how-to-format-curly-braces-for-autoformat-in-intellij">How to format curly braces for autoformat in intellij</a></h3>
<pre><code class="hljs language-javascript"><span class="hljs-comment">// 설정 전: import {CommonModule} from "@angular/common";</span>
<span class="hljs-keyword">import</span> { <span class="hljs-title class_">CommonModule</span> } <span class="hljs-keyword">from</span> <span class="hljs-string">"@angular/common"</span>;
</code></pre>
<p>설정 전을 보시면 <code>{}</code> 의 안쪽의 띄워쓰기가 없이 설정되어 있습니다.
이 부분을 설정하는 곳은!! <code>ES6 import/export braces</code> 입니다.</p>
<p><code>Settings > Editor > Code Style > Javascript > Spaces > ES6 import/export braces</code> 체크 하면 됩니다.</p>
<h3>🔖 추가로 Object 도 바로 위에 있습니다.</h3>
<pre><code class="hljs language-javascript"><span class="hljs-comment">// 설정 전: const {name, image} = getUser();</span>
<span class="hljs-keyword">const</span> { name, image } = <span class="hljs-title function_">getUser</span>();
</code></pre>
<p><code>Settings > Editor > Code Style > Javascript > Spaces > Object literal braces</code> 체크 하면 됩니다. 👍</p>
<h2>preserve whitespace in self closing tag</h2>
<h3>관련 링크: <a href="https://intellij-support.jetbrains.com/hc/en-us/community/posts/207060005-Code-formatting-preserve-whitespace-in-self-closing-tags">Code formatting - preserve whitespace in self closing tags</a></h3>
<pre><code class="hljs language-html"><span class="hljs-comment">&#x3C;!-- 설정 전: &#x3C;input type="text"/> --></span>
<span class="hljs-tag">&#x3C;<span class="hljs-name">input</span> <span class="hljs-attr">type</span>=<span class="hljs-string">"text"</span> /></span>
</code></pre>
<p>설정 전을 보시면 <code>/></code> 부분에 띄워쓰기가 되지 않는 걸로 되어 있습니다.
이 부분을 위한 설정은!! <code>in empty tag</code> 입니다.</p>
