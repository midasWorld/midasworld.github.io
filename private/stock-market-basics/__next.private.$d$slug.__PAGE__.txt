4:T7be6,<h2>주식이란?</h2>
<p>기업이 자금을 조달하기 위해 발행하는 <strong>소유권 증서</strong>. 주식을 사면 그 기업의 주주(공동 소유자)가 된다.</p>
<ul>
<li>기업 이익 일부를 <strong>배당</strong>으로 받을 권리</li>
<li>주가 상승 시 <strong>시세차익</strong> 실현 가능</li>
<li>주주총회 <strong>의결권</strong> 행사 가능 (소액주주는 사실상 의미 없음)</li>
</ul>

<h3>ETF 시장</h3>
<p>**ETF(Exchange Traded Fund)**란 특정 지수·자산을 추종하는 펀드를 주식처럼 거래소에서 사고파는 상품이다.</p>

<h4>괴리율 이해</h4>
<p>ETF는 **NAV(순자산가치)**와 <strong>시장가격</strong> 사이에 차이가 생길 수 있다.</p>
<pre><code class="hljs language-css">괴리율(%) = (ETF 시장가격 - <span class="hljs-selector-tag">NAV</span>) / <span class="hljs-selector-tag">NAV</span> × <span class="hljs-number">100</span>
</code></pre>
<ul>
<li>괴리율이 +2% 이상이면 ETF가 고평가 → 매수 시 불리</li>
<li>거래량 적은 ETF일수록 괴리율 벌어지기 쉬움</li>
<li><strong>KODEX 200</strong>, <strong>TIGER 미국S&#x26;P500</strong> 등 대형 ETF는 괴리율 거의 없음</li>
</ul>
<blockquote>
<p>실전 팁: 해외지수 추종 ETF(나스닥100, S&#x26;P500)는 NAV가 전날 종가 기준이라, 당일 해외시장이 크게 움직였다면 장 초반 괴리율이 일시적으로 벌어질 수 있다.</p>
</blockquote>

<h3>선물·옵션 기초 (개인투자자 필수 지식)</h3>
<p>직접 투자하지 않더라도 <strong>선물·옵션의 만기일 효과</strong>는 현물 주식에 영향을 주므로 반드시 알아야 한다.</p>
<h4>선물(Futures)</h4>
<ul>
<li>미래 특정 시점에 특정 가격으로 사고팔기로 약속하는 계약</li>
<li>KOSPI200 선물, 미니코스피200 선물 등</li>
<li><strong>레버리지</strong> 내재 → 증거금의 수배 손익 발생</li>
<li>개인이 직접 거래 시 원금 초과 손실 가능 → 초보자에게 비권장</li>
</ul>
<h4>옵션(Options)</h4>
<ul>
<li>특정 가격에 사거나(콜) 팔 수(풋) 있는 권리를 사고파는 계약</li>
<li><strong>콜옵션</strong>: 주가 상승 시 이익 / <strong>풋옵션</strong>: 주가 하락 시 이익</li>
<li>만기일에 가치가 0이 될 수 있는 소멸 상품</li>
</ul>

<p>없는 주식을 <strong>대주(빌려서)</strong> 먼저 팔고, 나중에 낮은 가격에 사서 갚아 차익을 남기는 투자 기법.</p>

</code></pre>

<li>하락에 베팅하는 수단이므로 시장에서 논란</li>
<li>기관·외국인은 가능하나 개인은 제한적 (증권사 대주서비스 이용)</li>

</blockquote>

<h2>주문 방법</h2>
<h3>호가창(매수호가/매도호가) 읽는 법</h3>
<pre><code class="hljs language-css"><span class="hljs-selector-attr">[매도잔량]</span>  <span class="hljs-selector-attr">[가격]</span>  <span class="hljs-selector-attr">[매수잔량]</span>
  <span class="hljs-number">3</span>,<span class="hljs-number">200</span>   <span class="hljs-number">70</span>,<span class="hljs-number">400</span>
  <span class="hljs-number">1</span>,<span class="hljs-number">800</span>   <span class="hljs-number">70</span>,<span class="hljs-number">300</span>
    <span class="hljs-number">900</span>   <span class="hljs-number">70</span>,<span class="hljs-number">200</span>    ← 현재가 근처
    <span class="hljs-number">500</span>   <span class="hljs-number">70</span>,<span class="hljs-number">100</span>
           <span class="hljs-number">70</span>,<span class="hljs-number">000</span>    <span class="hljs-number">2</span>,<span class="hljs-number">100</span>
           <span class="hljs-number">69</span>,<span class="hljs-number">900</span>    <span class="hljs-number">4</span>,<span class="hljs-number">500</span>
           <span class="hljs-number">69</span>,<span class="hljs-number">800</span>    <span class="hljs-number">8</span>,<span class="hljs-number">200</span>
</code></pre>
<ul>
<li><strong>매도호가(빨간색 위쪽)</strong>: 팔려는 사람들이 제시한 가격대. 낮을수록 매수 즉시 체결 가능</li>
<li><strong>매수호가(파란색 아래쪽)</strong>: 사려는 사람들이 제시한 가격대. 높을수록 매도 즉시 체결 가능</li>
<li><strong>스프레드</strong>: 최우선 매도호가와 최우선 매수호가 간 차이 → 좁을수록 유동성 좋음</li>
<li><strong>잔량이 많은 호가대</strong>: 심리적 지지/저항 역할. 대량 매도벽이 있으면 단기 돌파 어려움</li>
</ul>
<blockquote>
<p>실전 팁: 잔량이 갑자기 사라지는(호가 취소) 현상이 반복되면 시세 조종 가능성 주의. 실제 체결까지 이어지는 잔량이 진짜 수요/공급이다.</p>
</blockquote>

<h3>주문 종류</h3>
<h4>지정가 주문</h4>
<p>원하는 가격을 직접 지정. 그 가격이 될 때까지 체결 안 됨.</p>
<pre><code class="hljs">삼성전자를 70,000원에 사고 싶다
→ 70,000원 지정가 매수
→ 주가가 70,000원 이하로 내려올 때 체결
</code></pre>
<h4>시장가 주문</h4>
<p>현재 시장 가격으로 즉시 체결. 급하게 사거나 팔 때.</p>
<blockquote>
<p>변동성이 큰 종목은 시장가 사용 시 예상보다 불리한 가격에 체결될 수 있음 (슬리피지). 특히 KOSDAQ 소형주에서 두드러짐.</p>
</blockquote>
<h4>조건부 지정가</h4>
<p>장 중에는 지정가, 장 마감 시 미체결이면 시장가로 전환.</p>
<h4>최유리 지정가</h4>
<p>주문 즉시 상대방 최우선 호가로 체결. 시장가와 유사하나 한 호가 단위까지만.</p>
<h4>최우선 지정가</h4>
<p>주문 시점의 최우선 호가로 주문 자동 입력. 빠른 체결을 원하되 슬리피지를 제한하고 싶을 때 유용.</p>
<hr>
<h3>동시호가와 단일가 결정 방식</h3>
<h4>동시호가 원리</h4>
<p>장 전(08:30<del>09:00)과 장 후(15:30</del>15:40) 동시호가 시간에는 가격이 움직이지 않고 주문만 쌓인다.</p>
<p><strong>단일가 결정 알고리즘:</strong></p>
<ol>
<li>모든 매수·매도 주문 집계</li>
<li><strong>가장 많은 수량을 체결시킬 수 있는 가격</strong> 탐색</li>
<li>해당 가격으로 일괄 체결 (단일가)</li>
</ol>
<pre><code class="hljs language-markdown">매수 주문  |  가격   | 매도 주문
  5,000주  | 70,200
  3,000주  | 70,100
  2,000주  | 70,000  |  1,000주
  1,000주  | 69,900  |  3,000주
<span class="hljs-code">           | 69,800  |  5,000주
</span>
→ 70,000원에서 매수 2,000주 + 매도 1,000주 → 최대 체결 1,000주
→ 가격 더 내려서... 최대 체결가 탐색
</code></pre>
<blockquote>
<p>장 전 동시호가에서 대규모 매수 주문이 특정 가격대에 쌓이면 시초가가 그 부근에서 결정된다. 시초가 예측에 활용 가능.</p>
</blockquote>
<h4>시간외 단일가 (15:40~18:00)</h4>
<ul>
<li>10분 단위로 단일가 결정 (15:40, 15:50, ... 17:50)</li>
<li>정규장 종가의 ±10% 범위에서만 주문 가능</li>
<li>거래량 극히 적음 → 유동성 리스크 존재</li>
</ul>

<h3>스탑로스(손절) 주문 활용법</h3>
<h4>개념</h4>
<p>손실이 일정 수준을 넘으면 자동으로 매도하는 주문. 심리적 판단 실수를 방지하는 핵심 장치.</p>
<h4>한국 HTS/MTS에서의 구현 방법</h4>
<p>대부분의 증권사는 <strong>"조건부 주문"</strong> 또는 <strong>"자동주문"</strong> 기능으로 유사 기능 제공.</p>
<p><strong>키움 영웅문 기준:</strong></p>
<ol>
<li>주문 화면 → "조건주문" 탭</li>
<li>조건: "현재가 ≤ 67,000원"</li>
<li>액션: "시장가 매도 100주"</li>
<li>유효기간 설정 후 등록</li>
</ol>
<blockquote>
<p>주의: 키움 등 국내 HTS의 조건부 주문은 <strong>HTS가 켜져 있을 때만 작동</strong>. 서버 스탑로스가 아니므로 HTS 종료 시 무효.</p>
</blockquote>
<p><strong>토스증권 기준:</strong></p>
<ul>
<li>"가격 도달 시 매도" 기능: 현재가 기준 하락 % 또는 절대 가격 설정</li>
<li>서버 사이드 작동 → HTS 꺼져도 유효 (단, 시장가 체결)</li>
</ul>
<h4>스탑로스 설정 원칙</h4>
<ul>
<li>손절 기준을 <strong>매수 전에</strong> 미리 설정 (감정 배제)</li>
<li>일반적으로 매수가 대비 <strong>-5% ~ -10%</strong> 설정</li>
<li>변동성이 큰 KOSDAQ 성장주는 -10~-15%도 사용</li>
<li>지지선 바로 아래에 스탑을 설정하는 것이 기술적으로 유효</li>
</ul>

<h2>수익률 계산</h2>
<pre><code class="hljs language-scss">수익률(%) = (매도가 - 매수가) / 매수가 × <span class="hljs-number">100</span>

예) <span class="hljs-number">10</span>만원에 사서 <span class="hljs-number">13</span>만원에 팔면
→ (<span class="hljs-number">130</span>,<span class="hljs-number">000</span> - <span class="hljs-number">100</span>,<span class="hljs-number">000</span>) / <span class="hljs-number">100</span>,<span class="hljs-number">000</span> × <span class="hljs-number">100</span> = <span class="hljs-number">30%</span>
</code></pre>
<h3>실질 수익률 (세금·수수료 반영)</h3>
<pre><code class="hljs language-ini">실질 수익 = 매도금액 - 매수금액 - 수수료(매수+매도) - 증권거래세 - 양도세

예) 1,000만원어치 매수 → 1,300만원에 매도 (수수료 0.015% 가정)
- 매수 수수료: 10,000,000 × <span class="hljs-attr">0.00015</span> = <span class="hljs-number">1</span>,<span class="hljs-number">500</span>원
- 매도 수수료: 13,000,000 × <span class="hljs-attr">0.00015</span> = <span class="hljs-number">1</span>,<span class="hljs-number">950</span>원
- 증권거래세 (KOSPI): 13,000,000 × <span class="hljs-attr">0.0018</span> = <span class="hljs-number">23</span>,<span class="hljs-number">400</span>원
- 총 비용: 26,850원
- 순이익: 3,000,000 - 26,<span class="hljs-attr">850</span> = <span class="hljs-number">2</span>,<span class="hljs-number">973</span>,<span class="hljs-number">150</span>원
</code></pre>

<h3>해외주식 양도소득세</h3>
<p>해외주식은 소액이라도 양도차익에 세금이 부과된다.</p>
<pre><code class="hljs language-scss">과세표준 = 연간 해외주식 양도차익 합계 - <span class="hljs-number">250</span>만원 (기본공제)
납부세액 = 과세표준 × <span class="hljs-number">22%</span> (지방소득세 포함)

예) 연간 해외주식 수익 <span class="hljs-number">1</span>,<span class="hljs-number">250</span>만원
→ (<span class="hljs-number">1</span>,<span class="hljs-number">250</span>만원 - <span class="hljs-number">250</span>만원) × <span class="hljs-number">22%</span> = <span class="hljs-number">220</span>만원 납부
</code></pre>
<h4>절세 팁</h4>
<ul>
<li><strong>손실 상계</strong>: 같은 해에 발생한 해외주식 손실과 합산 가능
<ul>
<li>예) A종목 +500만원, B종목 -300만원 → 과세표준 200만원 → 250만원 공제 내 → 세금 0</li>
</ul>
</li>
<li><strong>연말 손실 실현</strong>: 12월 말 평가손실 종목을 매도해 당해 연도 이익과 상계</li>
<li><strong>매년 250만원 이하 분산</strong>: 연도별 250만원 공제 활용</li>
<li>신고 기한: <strong>다음 해 5월 종합소득세 신고 기간</strong>에 직접 신고 또는 증권사 대행 서비스 이용</li>
</ul>

<h3>해외주식 배당소득세</h3>
<p>미국 주식 배당은 <strong>미국에서 15% 원천징수</strong> 후 지급된다.</p>
<pre><code class="hljs language-bash">예) 애플(AAPL) 배당 <span class="hljs-variable">$100</span> 발생
→ 미국 원천징수 <span class="hljs-variable">$15</span>
→ 실수령 <span class="hljs-variable">$85</span>

국내 배당소득세율 14% → 미국에서 이미 15% 냈으므로
→ 추가 납부 없음 (외국납부세액공제 적용)
</code></pre>
<h4>금융소득종합과세 주의</h4>
<ul>
<li>국내외 배당·이자 합계가 연 <strong>2,000만원 초과</strong> 시 종합과세 대상</li>
<li>최고 세율 49.5%(지방세 포함)까지 적용 가능</li>
<li>배당 수입이 늘어날수록 절세 계좌(ISA, 연금저축) 활용 필수</li>
</ul>
