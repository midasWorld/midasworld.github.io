15:T7824,<h2>들어가며 — 왜 재무제표를 읽어야 하는가</h2>
<p>"이 기업은 유명하니까 좋겠지"는 투자가 아니다. 실제로 기업의 건강 상태를 확인하는 유일한 방법은 <strong>재무제표</strong>를 직접 읽는 것이다.</p>
<p>재무제표는 처음엔 어렵게 느껴지지만, 핵심 개념 몇 가지만 이해하면 30분 안에 기업의 재무 건전성을 파악할 수 있다.</p>
<pre><code class="hljs language-scss">재무제표 <span class="hljs-number">3</span>총사
─────────────────────────────────────────────────────────
  손익계산서          대차대조표          현금흐름표
(Income Statement) (Balance Sheet)  (Cash Flow Statement)
        │                 │                  │
   얼마 벌었나?    지금 얼마 갖고      실제 현금은
   (수익성)         있나? (안전성)      얼마인가?
                                      (현금창출력)
─────────────────────────────────────────────────────────
세 가지 모두 함께 봐야 기업의 진짜 상태가 보인다
</code></pre>
<blockquote>
<p><strong>💡 핵심 원칙</strong>: 순이익이 높아도 현금이 없으면 위험하다. 3가지 재무제표를 반드시 함께 봐야 한다.</p>
</blockquote>

<h2>2. 대차대조표 (Balance Sheet)</h2>
<p>대차대조표는 <strong>특정 시점에 기업이 무엇을 갖고 있고, 무엇을 빚졌는지</strong> 보여준다.</p>

<pre><code class="hljs language-scss">┌─────────────────────────────────────────────────────┐

<h2>3. 현금흐름표 (Cash Flow Statement)</h2>
<p>현금흐름표는 <strong>실제 현금이 어디서 들어오고 어디로 나갔는지</strong> 보여준다. 이것이 가장 중요한 재무제표라는 전문가도 많다.</p>
<blockquote>
<p><strong>💡 핵심</strong>: 순이익은 회계 기준으로 조작 가능하지만, 현금은 거짓말하지 않는다.</p>
</blockquote>
<h3>3-1. 현금흐름 3가지 유형</h3>
<pre><code class="hljs language-scss">현금흐름표 구분
─────────────────────────────────────────────────────────────────
영업활동 현금흐름 (Operating Cash Flow, OCF)
  → 본업에서 실제로 들어온 현금
  → "이 기업은 장사로 돈을 버는가?"
  → 항상 (+) 이어야 한다 (지속적 (-) 이면 적신호)

투자활동 현금흐름 (Investing Cash Flow)
  → 설비 투자, 자산 매각, 인수합병 등
  → 성장하는 기업은 보통 (-) (미래를 위해 투자 중)
  → 너무 공격적인 투자는 현금 부담 위험

재무활동 현금흐름 (Financing Cash Flow)
  → 주식 발행, 배당 지급, 차입/상환
  → 성숙 기업은 보통 (-) (자사주 매입, 배당 지급)
─────────────────────────────────────────────────────────────────
</code></pre>
<h3>3-2. FCF (잉여현금흐름) — 가장 중요한 지표</h3>
<pre><code class="hljs language-swift"><span class="hljs-type">Free</span> <span class="hljs-type">Cash</span> <span class="hljs-type">Flow</span> (<span class="hljs-type">FCF</span>) 계산
<span class="hljs-operator">────────────────────────────────────────────────</span>
<span class="hljs-type">FCF</span> <span class="hljs-operator">=</span> 영업활동 현금흐름 <span class="hljs-operator">-</span> 자본적 지출(<span class="hljs-type">CAPEX</span>)

예시:
  영업활동 현금흐름:  <span class="hljs-operator">+</span><span class="hljs-variable">$110</span>B
  <span class="hljs-type">CAPEX</span> (설비투자):   <span class="hljs-operator">-</span><span class="hljs-variable">$10</span>B
  <span class="hljs-operator">─────────────────────────</span>
  <span class="hljs-type">FCF</span>:               <span class="hljs-operator">+</span><span class="hljs-variable">$100</span>B  <span class="hljs-operator">←</span> 이 돈이 실제 여유 현금

이 <span class="hljs-type">FCF로</span> 기업은:
  <span class="hljs-operator">✓</span> 자사주 매입 (주주 환원)
  <span class="hljs-operator">✓</span> 배당 지급
  <span class="hljs-operator">✓</span> 빚 갚기
  <span class="hljs-operator">✓</span> 인수합병
  <span class="hljs-operator">✓</span> 현금 보유
<span class="hljs-operator">────────────────────────────────────────────────</span>
</code></pre>

<h2>7. 재무제표 함정 — 이런 기업 조심하라</h2>
<h3>🚨 레드 플래그 10가지</h3>
<pre><code class="hljs language-markdown"><span class="hljs-bullet">1.</span> 순이익은 증가하는데 FCF는 감소
   → 회계상 이익을 부풀릴 가능성 (수익 조기 인식 등)

<span class="hljs-bullet">2.</span> 재고가 매출보다 빠르게 증가
   → 물건이 안 팔리고 있는 신호

<span class="hljs-bullet">3.</span> 매출채권이 매출보다 빠르게 증가
   → 현금을 못 받고 있거나 공격적 회계 처리

<span class="hljs-bullet">4.</span> 반복적인 "일회성 비용" 처리
   → 매년 일회성 손실이 나오면 사실상 일상적 비용

<span class="hljs-bullet">5.</span> 감사 의견에 "계속기업 관련 중요한 불확실성"
   → 회사가 망할 수도 있다는 뜻

<span class="hljs-bullet">6.</span> 임원의 지속적 주식 매도
   → 내부자들이 미래를 부정적으로 본다는 신호

<span class="hljs-bullet">7.</span> 부채가 급격히 증가하면서 FCF는 정체
   → 빚으로 성장을 유지하는 구조

<span class="hljs-bullet">8.</span> 여러 번의 CFO/CEO 교체
   → 내부 문제가 있을 가능성

<span class="hljs-bullet">9.</span> 감사법인 교체
   → 이전 감사법인과 의견 충돌 가능성

<span class="hljs-bullet">10.</span> 관련 당사자 거래가 많음 (Related Party Transactions)
<span class="hljs-code">    → 오너 일가와의 불투명한 거래 가능성
</span></code></pre>
