<pre><code class="hljs language-scss">┌─────────────────────────────────────────────────────────────────────┐

<h3>1.2 금리 인상이 주가를 떨어뜨리는 이유 — DCF 모델</h3>
<pre><code class="hljs language-ini">┌─────────────────────────────────────────────────────────────────────┐
│          DCF(현금흐름할인) 모델로 이해하는 금리-주가 역관계          │
└─────────────────────────────────────────────────────────────────────┘

  주가 = 미래 이익들의 현재 가치 합계

  현재 가치 공식:
  <span class="hljs-attr">PV</span> = Future Cash Flow ÷ (<span class="hljs-number">1</span> + 할인율)^n

  ──────────────────────────────────────────────────────────────────

  사례: Apple이 향후 3년간 매년 $10 EPS 예상

  금리 3% (낮을 때):                금리 7% (높을 때):
  ─────────────────                 ─────────────────
  1년후 $10 → <span class="hljs-attr">PV</span> = <span class="hljs-number">10</span>/<span class="hljs-number">1.03</span>  = <span class="hljs-variable">$9</span>.<span class="hljs-number">71</span>  <span class="hljs-number">10</span>/<span class="hljs-number">1.07</span>  = <span class="hljs-variable">$9</span>.<span class="hljs-number">35</span>
  2년후 $10 → <span class="hljs-attr">PV</span> = <span class="hljs-number">10</span>/<span class="hljs-number">1.06</span>  = <span class="hljs-variable">$9</span>.<span class="hljs-number">43</span>  <span class="hljs-number">10</span>/<span class="hljs-number">1.14</span>  = <span class="hljs-variable">$8</span>.<span class="hljs-number">73</span>
  3년후 $10 → <span class="hljs-attr">PV</span> = <span class="hljs-number">10</span>/<span class="hljs-number">1.09</span>  = <span class="hljs-variable">$9</span>.<span class="hljs-number">15</span>  <span class="hljs-number">10</span>/<span class="hljs-number">1.23</span>  = <span class="hljs-variable">$8</span>.<span class="hljs-number">16</span>
                            ──────                  ──────
              합계 현재가치: $28.29                  $26.24

  금리 3% → 7%  :  주가 이론값 7.2% 하락!

  ──────────────────────────────────────────────────────────────────

  특히 성장주(미래 이익 비중 높음)에 타격 더 큼!

  가치주: 이미 이익 내는 회사 → 금리 영향 상대적 작음
  성장주: 10년 후 이익 기대 → 할인율 상승 시 현재가치 폭락!

         금리 인상
            │
     ┌──────┴──────┐
     │             │
  성장주           가치주
  -30% ~ -50%    -10% ~ -20%
  (TSLA, NVDA)   (JPM, XOM)
</code></pre>

<h3>3.3 한국 투자자의 이중 환율 리스크</h3>
<pre><code class="hljs language-bash">┌─────────────────────────────────────────────────────────────────────┐
│               한국 투자자만의 이중 환율 리스크                       │
└─────────────────────────────────────────────────────────────────────┘

  한국 투자자가 Apple 주식 보유 시:

  수익/손실 = 주가 변동 × 환율 변동

  시나리오 A: 주가 상승 + 달러 약세 (최악)
  ───────────────────────────────────────
  AAPL: <span class="hljs-variable">$150</span> → <span class="hljs-variable">$165</span> (+10%)
  USD/KRW: 1,300 → 1,200 (-7.7% 원화 강세)
  원화 기준 실제 수익:
    매수: <span class="hljs-variable">$150</span> × 1,300 = ₩195,000
    매도: <span class="hljs-variable">$165</span> × 1,200 = ₩198,000
    수익: +₩3,000 (= +1.5%)  ← 주가는 10% 올랐는데!

  시나리오 B: 주가 상승 + 달러 강세 (최고)
  ───────────────────────────────────────
  AAPL: <span class="hljs-variable">$150</span> → <span class="hljs-variable">$165</span> (+10%)
  USD/KRW: 1,300 → 1,400 (+7.7% 원화 약세)
  원화 기준 실제 수익:
    매수: <span class="hljs-variable">$150</span> × 1,300 = ₩195,000
    매도: <span class="hljs-variable">$165</span> × 1,400 = ₩231,000
    수익: +₩36,000 (= +18.5%)  ← 주가는 10%인데 18.5% 수익!

  시나리오 C: 주가 하락 + 달러 강세
  ───────────────────────────────────
  AAPL: <span class="hljs-variable">$150</span> → <span class="hljs-variable">$135</span> (-10%)

<h3>5.2 지표별 주가 반응 매트릭스</h3>
<pre><code class="hljs language-scss">┌─────────────────────────────────────────────────────────────────────┐
│           경제 지표 발표 결과별 주가 반응                            │
└─────────────────────────────────────────────────────────────────────┘

  ┌──────────────────┬───────────────────────┬───────────────────────┐
  │ 지표              │ 예상보다 강하게 나옴  │ 예상보다 약하게 나옴  │
  ├──────────────────┼───────────────────────┼───────────────────────┤
  │ NFP (고용)        │ 혼재                  │ 혼재                  │
  │                  │ 경제 좋음 ↑           │ 경기 침체 우려 ↓      │
  │                  │ But 금리 인상 우려 ↓  │ But 금리 인하 기대 ↑  │
  ├──────────────────┼───────────────────────┼───────────────────────┤
  │ CPI (물가)        │ 주식 ↓↓              │ 주식 ↑↑               │
  │                  │ 금리 인상 우려 증가   │ 금리 인하 기대 증가   │
  ├──────────────────┼───────────────────────┼───────────────────────┤
  │ GDP               │ 주식 ↑               │ 주식 ↓                │
  │                  │ 경제 성장 좋은 신호   │ 경기 침체 우려        │
  ├──────────────────┼───────────────────────┼───────────────────────┤
  │ 소매판매          │ 주식 ↑               │ 주식 ↓                │
  │                  │ 소비 건전 신호        │ 소비 감소 우려        │
  ├──────────────────┼───────────────────────┼───────────────────────┤
  │ ISM PMI           │ 주식 ↑               │ 주식 ↓                │
  │                  │ (<span class="hljs-number">50</span> 이상 = 확장)      │ (<span class="hljs-number">50</span> 이하 = 수축)      │
  └──────────────────┴───────────────────────┴───────────────────────┘

  ⚠️  고용지표는 '역설'이 있다:
  고용이 너무 좋음 → 연준이 금리 내리지 않겠다 → 주식 하락
  고용이 약함    → 연준이 금리 내릴 것 → 주식 상승
  (<span class="hljs-number">2023</span>~<span class="hljs-number">2024</span>년 이 패턴 반복됨)
</code></pre>
<blockquote>
<p><strong>실전 체크포인트</strong></p>
<p>경제 지표 발표일 전날은 포지션을 크게 늘리지 않는 것이 좋다. 특히 CPI와 FOMC는 시장 변동성을 극도로 높인다. Investing.com의 "경제 캘린더"를 북마크해두자.</p>
</blockquote>

<h3>6.2 EPS 서프라이즈 vs 실적 쇼크</h3>
<pre><code class="hljs language-bash">┌─────────────────────────────────────────────────────────────────────┐
│                  실적 발표 결과 유형별 반응                           │
└─────────────────────────────────────────────────────────────────────┘

  EPS (Earnings Per Share) = 주당순이익
  <span class="hljs-string">"컨센서스"</span> = 애널리스트 평균 예상치

  유형 1: 어닝 서프라이즈 (Beat)
  ─────────────────────────────
  예상: <span class="hljs-variable">$1</span>.50 EPS
  실제: <span class="hljs-variable">$1</span>.80 EPS (+20% 상회)
  → 즉각 5~15% 주가 상승 (애프터마켓)

  유형 2: 인라인 (In-line)
  ─────────────────────────
  예상: <span class="hljs-variable">$1</span>.50 EPS
  실제: <span class="hljs-variable">$1</span>.52 EPS (딱 부합)
  → 큰 변동 없음 (이미 가격에 반영됨)

  유형 3: 어닝 쇼크 (Miss)
  ─────────────────────────
  예상: <span class="hljs-variable">$1</span>.50 EPS
  실제: <span class="hljs-variable">$1</span>.20 EPS (-20% 하회)
  → 즉각 10~30% 주가 하락

  ──────────────────────────────────────────────────────────────────

  ⚠️  <span class="hljs-string">"실적은 좋은데 주가는 왜 떨어져?"</span> 현상

  이유: 실적 발표 전에 이미 주가가 기대감으로 올라있었음
  <span class="hljs-string">"Buy the rumor, Sell the news"</span> (소문에 사서 뉴스에 팔아라)

  가이던스(Guidance)가 더 중요:
  Q2 실적이 좋아도 Q3 전망이 나쁘면 → 주가 하락
  Q2 실적이 나빠도 Q3 전망이 좋으면 → 주가 상승
</code></pre>

<h3>7.2 방어주 vs 성장주의 금리 민감도</h3>
<pre><code class="hljs language-markdown">┌─────────────────────────────────────────────────────────────────────┐
│              방어주 vs 성장주 금리 민감도 비교                       │
└─────────────────────────────────────────────────────────────────────┘

  방어주 특징:
  ──────────
<span class="hljs-bullet">  -</span> 경기 무관 꾸준한 이익 (음식, 약, 전기)
<span class="hljs-bullet">  -</span> 높은 배당률 (3~5%)
<span class="hljs-bullet">  -</span> 금리 상승 시: 채권 대비 매력도 하락 → 주가 하락
<span class="hljs-bullet">  -</span> 금리 하락 시: 채권 대비 매력도 상승 → 주가 상승
<span class="hljs-bullet">  -</span> 예: KO (코카콜라), PG (P&#x26;G), JNJ (존슨앤존슨)

  성장주 특징:
  ────────────
<span class="hljs-bullet">  -</span> 현재 이익보다 미래 성장 기대
<span class="hljs-bullet">  -</span> 낮은 배당(또는 무배당)
<span class="hljs-bullet">  -</span> 금리 상승 시: DCF 할인율 급상승 → 주가 폭락
<span class="hljs-bullet">  -</span> 금리 하락 시: 미래 이익 현재가치 급상승 → 주가 폭등
<span class="hljs-bullet">  -</span> 예: NVDA, TSLA, AMZN, META

  금리 인상 1%p당 예상 주가 영향:
  방어주 (유틸리티, 필수소비재): -5% ~ -10%
  가치주 (금융, 에너지):          -2% ~ +5%
  성장주 (기술, 임의소비재):     -15% ~ -30%

  ──────────────────────────────────────────────────────────────────

  실전 응용:

  연준 금리 인상 사이클 진입 신호 시
  → 성장주 비중 줄이고 가치주·에너지 비중 확대

  연준 금리 인하 신호 (or 경기 침체 공포) 시
  → 방어주·채권 비중 확대, 이후 성장주 재편입
</code></pre>
<blockquote>
<p><strong>실전 체크포인트</strong></p>
<p>섹터 로테이션을 실행하는 가장 간단한 방법: SPDR의 섹터 ETF (XLK, XLV, XLF 등)를 활용해 경기 사이클에 맞게 비중을 조절한다. 매월 각 섹터 ETF의 상대적 강도를 비교해보자.</p>
</blockquote>

<p>VIX를 매일 체크하는 습관을 들이자. Google에서 "CBOE VIX" 검색하거나, 증권사 앱에서 VIX 시세를 확인할 수 있다. VIX가 20을 넘으면 조심하고, 30을 넘으면 분할 매수 계획을 세우기 시작하자.</p>

<h2>9. 모든 것을 통합하는 투자 프레임워크</h2>
<h3>9.1 거시경제 + 지정학 종합 판단 체계</h3>
<pre><code class="hljs">┌─────────────────────────────────────────────────────────────────────┐
│               거시경제 종합 판단 대시보드                            │
└─────────────────────────────────────────────────────────────────────┘

  매월 체크해야 할 7가지 지표:

  지표 1. 연준 금리 방향성
  ─────────────────────────
  □ 금리 인상 사이클 → 성장주 비중 축소, 가치주·배당주 확대
  □ 금리 동결 → 현 포트폴리오 유지
  □ 금리 인하 사이클 → 성장주·장기채 비중 확대

  지표 2. CPI 트렌드
  ─────────────────────────
  □ CPI 상승 중 → 에너지·원자재·TIPS 비중 확대
  □ CPI 하락 중 → 기술주·성장주 비중 확대 시작

  지표 3. 경기선행지수 (LEI)
  ─────────────────────────
  □ 3개월 연속 하락 → 방어적 포트폴리오로 전환
  □ 반등 신호 → 경기민감주(산업재, 기술주) 선취매

  지표 4. 달러 인덱스 (DXY)
  ─────────────────────────
  □ DXY 상승 → 다국적 기업 실적 주의, 원자재 약세 가능
  □ DXY 하락 → 신흥국, 원자재, 다국적 기업 수혜

  지표 5. VIX 레벨
  ─────────────────────────
  □ VIX &#x3C; 20 → 정상 투자
  □ VIX 20~30 → 일부 현금 확보
  □ VIX > 30 → 분할 매수 시작
  □ VIX > 40 → 적극 매수 (역발상)

  지표 6. 지정학 리스크 온도
  ─────────────────────────
  □ 고조 → 에너지·방산·금 비중 확대
  □ 완화 → 기술·소비주로 이동

  지표 7. 실적 시즌 컨센서스
  ─────────────────────────
  □ 어닝 서프라이즈 연속 → 강세 신호
  □ 가이던스 하향 연속 → 매도 준비
</code></pre>
<h3>9.2 초보 투자자를 위한 단순화된 규칙</h3>
<pre><code class="hljs language-scss">┌─────────────────────────────────────────────────────────────────────┐
│                  초보자 거시경제 대응 <span class="hljs-number">3</span>원칙                           │
└─────────────────────────────────────────────────────────────────────┘

  원칙 <span class="hljs-number">1</span>. 연준을 거스르지 마라 (Don't Fight the Fed)
  ──────────────────────────────────────────────────
  연준이 금리 올리는 동안 → 성장주 묻지마 매수 금지
  연준이 금리 내리는 동안 → 성장주 보유 및 추가 매수 긍정적

  원칙 <span class="hljs-number">2</span>. VIX <span class="hljs-number">40</span> 이상 = 매수 기회
  ──────────────────────────────────────────────────
  패닉이 정점일 때가 실은 매수 적기
  단, 반드시 분할 매수 (전체 자금의 <span class="hljs-number">20%</span>씩 <span class="hljs-number">4</span>번 나눠서)

  원칙 <span class="hljs-number">3</span>. 지정학 이슈로 급락한 펀더멘털 좋은 기업 = 기회
  ──────────────────────────────────────────────────
  전쟁이나 제재로 주가가 급락해도
  해당 기업의 사업 자체가 타격받지 않았다면
  → 단기 과잉 반응 → 매수 기회

  <span class="hljs-number">2022</span>년 NVDA 미중 갈등 급락 → <span class="hljs-number">2023</span>년 +<span class="hljs-number">239%</span> 사례가 대표적
</code></pre>

<h2>10. 시리즈 마무리</h2>
<pre><code class="hljs language-sql">┌─────────────────────────────────────────────────────────────────────┐
│              미국 주식 투자 입문 시리즈 완성                         │

  ✅ <span class="hljs-number">01</span>편: 계좌 개설부터 첫 주문까지

  ✅ <span class="hljs-number">03</span>편: 어떤 주식 살까 — 기업 분석 첫걸음

  ✅ <span class="hljs-number">09</span>편: 매수<span class="hljs-operator">/</span>매도 타이밍
  ✅ <span class="hljs-number">10</span>편: 포트폴리오 구성과 리밸런싱
  ✅ <span class="hljs-number">11</span>편: 거시경제와 국제 시사가 주가를 움직인다 (지금 읽는 중)

  ──────────────────────────────────────────────────────────────────

  <span class="hljs-number">11</span>편을 마치며:

  주식 투자는 결국 "세상이 어떻게 돌아가는지" 이해하는 것이다.
  연준 의장의 발언 한 마디, 러시아의 군사 행동, 중국의 반도체 정책,
  이 모든 것이 당신이 보유한 Apple, NVIDIA, Microsoft 주가에 연결되어 있다.

  뉴스를 그냥 뉴스로 보지 말고,
  "이게 어떤 섹터에 유리한가? 어떤 섹터에 불리한가?"를
  항상 생각하는 습관을 들이자.

  그것이 아마추어와 프로를 가르는 가장 중요한 차이다.
